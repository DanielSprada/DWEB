HYPERLINK

Hyperlink é uma referência contida em um documento em hypertexto.

Os hyperlinks fazem um "apontamento", seja para outras partes de um mesmo documento, para outra página ou para um download, dentre outras.

Sintaxe do Hyperlink:

<a href="endereço">Conteúdo Referenciado Clicável</a>

Também é importante utilizar o comando "target" para especificar como será a abertura do hyperlink _blank será aberta uma nova guia e _self abrirá na guia atual.

Importante também escrever sobre o que é referente o link com o comando "rel".

TIPOS DE HYPERLINK

Externos ou Remotos

São links que fazem referência a sites e páginas na internet.
Na propriedade "href" utilizamos a URL (Uniform Resource Locator) do site.

Hyperlinks Internos

Apontam para um lugar diferente na mesma página. PErmite acessar diferentes parágrafos ou seções o que facilita a utilização quando o documento é muito extenso.

Sua sintaxe é: <a href="#referencia">Conteúdo Referenciado</a>
<a name="referencia"></a> - Essa parte vai ser escrita na parte do conteúdo onde você quer ir ao clicar no hyperlink.

Hiperlinks Locais

Apontam para outras páginas dentro do mesmo site. Isso acontece muito quando navegamos, clicamos em um link e carrega uma nova página no site.

Sua sintaxe se dá da mesma forma das demais, apenas mudando o conteúdo da href q será: href="arquivo.html"

Hyperlinks em Imagens

É possível colocar hyperlinks em imagens, envolvendo a tag de imagem na de hyperlink.

Sintaxe: <a href="referencia"><img src="imagem.jpg"></a>

Hyperlinks para Download de Arquivos

São links que apontam para arquivos e permite que seja feito download dos mesmos.

A sintese básica é a mesma adicionando o comando download="arquivo.extensão" e type="tipo".

Hyperlinks para Endereço Eletrônico

São links que apontam para um e-mail. Ao clicar irá, em geral, executar o aplicativo de e-mail padrão do sistema/usuário.

Na sintese, dentro do haref irá incluir mailto:. Como por exemplo: href="mailto:email@email.com".
